# Bycar (AV.BY Clone)

**Bycar** представляет собой высокотехнологичный специализированный RESTful API сервис, разработанный как комплексное решение для автоматизации и управления полным жизненным циклом объявлений в сегменте купли-продажи транспортных средств. Архитектура системы спроектирована по принципу современного цифрового маркетплейса, где ключевое внимание уделяется обеспечению строгой консистентности и целостности данных при обработке многоуровневых связей. Сервис эффективно оперирует сложными техническими спецификациями автомобилей (включая уникальные идентификаторы, пробег и комплектации), интегрирует детализированные профили пользователей и поддерживает иерархическую структуру справочников (марка — модель — поколение). Благодаря четкому разделению зон ответственности и применению современных паттернов проектирования, система гарантирует высокую надежность хранения данных и предоставляет гибкий интерфейс для масштабирования функционала торговой площадки.
## Технологический стек:
* **Java 17+**
* **Spring Boot 3.x** (Web, Data JPA, Validation)
* **Maven** (Сборка и управление зависимостями)
* **PostgreSQL** (Реляционная база данных)
* **Hibernate** (ORM для работы с БД)
* **Docker / Docker Compose** (Контейнеризация БД)
* **GitHub Actions** (CI/CD пайплайны)
* **SonarCloud** (Статический анализ качества кода)
