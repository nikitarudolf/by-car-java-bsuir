# Bycar (AV.BY Clone)
Bycar — это RESTful API сервис для размещения и управления объявлениями о продаже автомобилей. Проект реализует полноценную серверную логику маркетплейса с учетом сложных связей между техническими характеристиками авто, пользователями и справочными данными.
## Технологический стек:
* Java 17+
* Spring Boot 3.x
* Maven / Gradle
* PostgreSQL

## Как запустить:
1. Клонировать репозиторий.
2. Выполнить ./mvnw spring-boot:run (для Maven).

## Реализованный функционал (Лаба 1):
* CRUD операции для сущности Advertisement (Объявления).
* Реализованы слои: Controller -> Service -> Repository.
* Валидация данных и маппинг Entity -> DTO.
