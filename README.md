# bycar (AV.BY Clone) - Учебный проект по Java
Данный проект является аналогом площадки для размещения объявлений о продаже автомобилей.

## Технологический стек:
* Java 17+
* Spring Boot 3.x
* Maven / Gradle
* PostgreSQL

## Как запустить:
1. Клонировать репозиторий.
2. Выполнить ./mvnw spring-boot:run (для Maven).

## Реализованный функционал (Лаба 1):
* CRUD операции для сущности Advertisement (Объявления).
* Реализованы слои: Controller -> Service -> Repository.
* Валидация данных и маппинг Entity -> DTO.
