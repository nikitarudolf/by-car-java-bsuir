# Bycar (AV.BY Clone)

**Bycar** — это специализированный RESTful API сервис, спроектированный для управления полным циклом размещения объявлений о продаже автомобилей. Система реализует архитектуру маркетплейса, обеспечивая консистентность данных при работе со сложными техническими характеристиками транспортных средств, профилями пользователей и иерархическими справочниками моделей.
## Технологический стек:
* **Java 17+**
* **Spring Boot 3.x** (Web, Data JPA, Validation)
* **Maven** (Сборка и управление зависимостями)
* **PostgreSQL** (Реляционная база данных)
* **Hibernate** (ORM для работы с БД)
* **Docker / Docker Compose** (Контейнеризация БД)
* **GitHub Actions** (CI/CD пайплайны)
* **SonarCloud** (Статический анализ качества кода)
