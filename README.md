# Bycar (AV.BY Clone)
Bycar — это высокопроизводительный RESTful API сервис, вдохновленный платформой av.by. Проект спроектирован как масштабируемая система для управления объявлениями о продаже транспортных средств, учитывающая сложные связи между техническими характеристиками авто, пользователями и рыночными данными.
## Технологический стек:
* Java 17+
* Spring Boot 3.x
* Maven / Gradle
* PostgreSQL

## Как запустить:
1. Клонировать репозиторий.
2. Выполнить ./mvnw spring-boot:run (для Maven).

## Реализованный функционал (Лаба 1):
* CRUD операции для сущности Advertisement (Объявления).
* Реализованы слои: Controller -> Service -> Repository.
* Валидация данных и маппинг Entity -> DTO.
